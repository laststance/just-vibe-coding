# Changelog

All notable changes to the "Just Vibe Coding" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2026-01-29

### Added

- ▶ Run and ◼ Stop buttons in editor title bar for vibe files
- Right-click context menu with "Run File" option
- Running state tracking with automatic button toggling

## [0.3.0] - 2026-01-29

### Added

- Python support (`vibe.py`) with python3 execution

## [0.2.0] - 2026-01-28

### Added

- Explorer view showing all past sessions

### Removed

- TypeScript (.ts) support — simplified to JavaScript only
- Reference pane feature (VS Code Copilot interference)

## [0.1.0] - 2026-01-26

### Added

- Initial release
- `Just Vibe Coding: Start` command to create isolated coding sessions
- Auto-execution on save with Node.js 22+ native TypeScript support
- Output channel for console feedback
- Status bar indicator "~ vibe coding ~"
- Session directory at `~/.just-vibe-coding/sessions/`
